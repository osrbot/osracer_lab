# osracer_lab

IsaacLab adapter for OSRacer.

This repository keeps simulation assets, tasks, and RL configs separate from the ROS 2 robot stack in `osrbot/osracer`.

## Layout

```text
source/osracer_lab_assets
source/osracer_lab_tasks
source/osracer_lab_rl
scripts
```
